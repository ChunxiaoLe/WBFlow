Testing demo for the paper: " WBFlow: Few-shot White Balance for sRGB Images via Reversible Neural Flows"

-------------------------------------------
The pre-trained model can be loaded from https://drive.google.com/file/d/1wqBCoBUQUvfaKeyT5_o0QvP82m8CDCb-/view?usp=sharing and stored in "./model"

-------------------------------------------
"Evalute_camera.py": 
	You can run this code to compare the white balance effects for the sRGB images from eight cameras.
        The indexes of testing sRGB images are in "./fold".

-------------------------------------------
"Dataset":
        Our rendered multi-camera sRGB dataset can be loaded from https://drive.google.com/drive/folders/1eX_hHxeMI6sKJu3De6P9Lunnz0i8P8aF?usp=share_link.
--------------------------------------------
